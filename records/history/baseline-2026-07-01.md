# 第一周历史基线补全

本周不是普通周报，而是资料库第一周，因此补齐 2017-2026 年间与公司业务强相关的历史资料。

## 为什么要补历史

公司做的是中小学AI素养评价，不是泛泛的AI教育资讯。要给售后、推广、内容同学使用，必须先建立一条政策和理论链路：

1. 国家战略提出发展智能教育。
2. 教育信息化和教育数字化逐步推进。
3. 教育评价改革明确强化过程评价、综合评价和五育导向。
4. 信息科技课程标准把AI、计算思维、数字化学习与创新纳入课程体系。
5. 中小学AI教育政策明确要求常态化教学与评价。
6. 地方政策开始落到五育融合、综合素质评价、成长数字档案袋。

## 公司业务链路

```text
AI教育政策
  -> 学校开设AI课程和实践活动
  -> 学生在真实任务中产生过程性表现
  -> 教师需要低负担记录
  -> AI辅助整理、分析、归类
  -> 映射到AI素养与五育指标
  -> 支撑综合素质评价、成长档案、售后汇报、学校展示
```

## 本周判断

AI教育的落地重点正在从“有没有课程”转向“如何评价学生AI素养”。对公司来说，最应该强调的不是“AI自动评价”，而是：

- AI辅助教师记录
- 过程性成长证据
- AI素养发展画像
- 五育指标映射
- 服务教师专业判断
